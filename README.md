# insomnia-plugin-firebase

Will return firebase user uid and token.

## Setup

Add the following attribute to the environment variables:
```json
{
  "firebase": {
    "config": {
      "apiKey": "",
      "authDomain": "",
      "databaseURL": "",
      "projectId": "",
      "storageBucket": "",
      "messagingSenderId": "",
      "appId": "",
    }
  }
}
```
